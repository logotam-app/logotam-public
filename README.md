## 🚀 Key Features

*   **OmniSearch™:** A centralized, privacy-first search bar routing queries directly to search engines and AI language models with zero intermediary tracking.
*   **Zero Tracking:** No shadow profiling, no targeted ads, and no unnecessary cookies. Pure utility.
*   **Blazingly Fast:** Built without heavy JS frameworks for instant loading on any mobile network.
*   **Local Focus:** Curated, verified links specific to each country's digital ecosystem (Finance, News, Auto, Jobs, Travel, etc.).
